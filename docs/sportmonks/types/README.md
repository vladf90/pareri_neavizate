# Types overview (export)

This folder contains an export of `Types_overview API V3.xlsx` so you can use it without Excel.

- Source: `Types_overview API V3.xlsx`
- Export: `types_overview_api_v3.csv`

## Notes
- The XLSX contains type IDs and code/name fields used across endpoints (events, statistics, standings, etc.).
- It generally **does not** include rich human descriptions for each type.
